## autoytdlp.vsh

It runs `yt-dlp` in batch with opinionated config (at `~/.config/autoytdlp.json`).

Example config:

```json
[
  {
    "id": "PL89gW297DWyDiqy6mbVmICA6D40H-6_cu",
    "path": "/home/me/Videos/muics"
  },
  {
    "id": "PL89gW297DWyD5EmTcMCNYDgs6u0cR7n_w",
    "path": "/home/me/Videos/muics/e"
  }
]
```
